# jupyter-base-notebook

## Usage

```
docker pull qwasfun/jupyter-base-notebook
```

```
docker run -p 8888:8888 qwasfun/jupyter-base-notebook
```

## feature

- add `C` kernel
- add  `JavaScript`, `TypeScript` kernel
- add Git extension
- add Chinese language pack
